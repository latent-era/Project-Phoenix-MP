2. In PortalApiClient.kt, add a new method in the "Sync Endpoints" section:

```kotlin
suspend fun pushPortalPayload(payload: PortalSyncPayload): Result<PortalSyncPushResponse> {
    return authenticatedRequest { token ->
        httpClient.post("${supabaseConfig.url}/functions/v1/mobile-sync-push") {
            bearerAuth(token)
            header("apikey", supabaseConfig.anonKey)
            setBody(payload)
        }
    }
}
```

CRITICAL: The `apikey` header must be present alongside the Bearer token. Supabase Edge Functions require BOTH. Without apikey, the Edge Function returns 401 even with a valid token.
